ALX Bug Tracker API

Report and solve bugs programmatically. This is the public API root for developers integrating with the service.

Docs & Schema

  • Swagger UI (interactive docs)
  • ReDoc
  • OpenAPI schema (JSON)

Health & Status

Use /health/ for uptime checks. It returns {"status": "ok"}.

API root for endpoints: /api/

Quickstart

Anonymous GETs are available for public resources. To create or modify resources, authenticate using token auth:

# Obtain token (example)
curl -X POST -d "username=you&password=YOUR_PASS" https://your-domain.com/api-token-auth/
# Use token
curl -H "Authorization: Token " -H "Content-Type: application/json" -d '{"title":"Bug","description":"..."}' https://your-domain.com/api/bug-post/
